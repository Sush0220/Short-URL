# MEAN Stack URL Shortener Project

This project is a URL shortener built using the MEAN stack (MongoDB, Express.js, AngularJS, Node.js).

## Overview

The MEAN stack URL shortener allows users to shorten long URLs into shorter, more manageable links. It provides a simple interface for users to input a long URL and receive a shortened version, which they can then share or use.

## Features

- Shorten long URLs into a unique, short code
- Redirect users from the short URL to the original long URL
- Store mappings between short URLs and their corresponding long URLs in a MongoDB database
- Track usage statistics, such as the number of times a short URL has been accessed

## Technologies Used

- MongoDB: A NoSQL database used to store the mappings between short URLs and their corresponding long URLs.
- Express.js: A web application framework for Node.js used to build the backend server for handling HTTP requests.
- AngularJS: A JavaScript framework used to build the frontend interface for the URL shortener.
- Node.js: A JavaScript runtime environment used to execute server-side code for handling HTTP requests and interacting with the MongoDB database.

## Installation

1. Clone the repository: `git clone <repository-url>`
2. Install dependencies: `npm install`
3. Start the server: `npm start`

## Usage

1. Navigate to the homepage of the URL shortener.
2. Enter a long URL that you want to shorten into the input field.
3. Click the "Shorten" button to generate a shortened version of the URL.
4. Copy the shortened URL and use it as needed.
